This is a rough draft of my thoughts during competitor exploration.

## Competitors analysed

- https://pomodoro-tracker.com/
- https://pomofocus.io/
- https://pomodorotimer.online/pt/
- https://pomodor.app/timer

## Features they have (that I haven't planned yet):

- Browser notification (pretty much all of them), but I don't like the way they ask for notifications
- Shortcuts (from https://pomofocus.io/)
- Global ranking (Focus time per week) -> Very good for gamification and engagement! Pomofocus does that but it's a bit hidden, could appear in more ways. Achievements/badges would also be cool.
- Customization: options like enable/disable auto-start, alarm sound, focus sounds (white noise, ticking sound), theme customizat
- Integrations ([todoist](https://www.todoist.com/), webhook. Would be cool to integrate with several task managers like jira, zapier, linear, etc)
- I18N
- Categories: presets with different sessions settings.
- Daily challenge: set a goal of time working or tasks achieved.
- Alarm volume

## Other notes

- All sites I've checked provide a poor UI and lack UX. I've noticed some basic customizations (like changing the pomodore/break time) require the user to open a modal, when on our site the user just clicks on the number and edits right there. But they're far more complete, with integrations and more granular customization. 
- For future considerations: add a streak system, to keep the user engaged (?)
- Other than using for work, people may use it to study it. Is there anything different we could do for this usecase? What are OTHER usecases?
- For a veery further future: maybe analyse the user patterns and give insights/suggestions on it. The system could detect if the user attention span increases over time, for example. Or notice the sweet spot of work/break based on their behaviour. The challenge might be how to track behaviour, since we don't have access to other tabs, unless we create an extension. So this increases scope a lot, but it's interesting.
- The app's UI could be more fluid, with nice background effects, etc. Or be customizable.

## Features requested on feedback pages

- cross-device sync (tasks, time, etc)
- gamification: badges, ranking, points, streaks. Duolingo is a great example of heavy gamification.
